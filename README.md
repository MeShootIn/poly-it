# Poly-IT

University project on the subject \"Internet technologies\" (first semester).

# CLI

* Install all modules:
```
npm install
```

* Run:
```
npm run start
```

* Run in dev-mode:
```
npm run dev
```

* Run all migrations and seed all files:
```
npm run db:init
```

* Rollback all migrations:
```
npm run db:rollback
```
